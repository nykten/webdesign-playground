# Nykten's Web Design Playground

## Dotted grid CSS-only background

Inspired by Spark AR's patch editor, this is how to replicate the design of the background of the patch editor, which is just dotted grids.

Implementation inspired from: [this codepen](https://codepen.io/ppuno/pen/GWVqxd) by Patricia Puno.

Good memories.
Now they live and seen in many node-based editors such as n8n, Supabase, Nodetoy and more.
